# Multi-Exposure-HDR-Fusion
An end-to-end HDR imaging pipeline using OpenCV. Implements MTB alignment, classical radiance reconstruction (Debevec &amp; Robertson), and enhanced Mertens Exposure Fusion with CLAHE.
# Multi-Exposure HDR Image Reconstruction and Enhancement

This project was prepared for the Image Processing course at İzmir Demokrasi Üniversitesi. The main focus of the project is to reduce detail loss in high-contrast scenes by combining multiple exposure images of the same scene.

## Project Objective

Standard camera sensors have a limited dynamic range. Because of this limitation, a single exposure image may not represent both dark and bright regions clearly. In a short exposure image, bright regions are usually preserved, but shadow details may be lost. In a long exposure image, dark regions become more visible, but bright regions may become overexposed.

The aim of this project is to combine three exposure images of the same scene and obtain a more balanced output. The selected input images are:

- `dark.jpg` - short exposure image
- `normal.jpg` - normal exposure image
- `bright.jpg` - long exposure image

These images are used as input exposure images. The final HDR and fusion results are generated by the implemented image processing methods.

## Method Overview

The project follows a multi-step image processing pipeline.

### 1. Image Loading and Preprocessing

The three input images are loaded using OpenCV. Before applying HDR reconstruction and fusion methods, the images are resized to the same resolution. This is necessary because multi-exposure merging methods require input images with identical dimensions.

### 2. MTB Alignment

Small shifts between exposure images may cause blur or ghost-like artifacts after merging. To reduce this problem, the Median Threshold Bitmap (MTB) alignment method is applied before HDR reconstruction and exposure fusion.

### 3. HDR Reconstruction

Two classical HDR reconstruction methods are used:

- Debevec HDR Merging
- Robertson HDR Reconstruction

Both methods use multiple exposure images and exposure time information to estimate an HDR representation of the scene.

### 4. Reinhard Tone Mapping

HDR outputs cannot be displayed directly on standard screens. Therefore, Reinhard tone mapping is applied to the Debevec and Robertson HDR results. This step converts HDR images into displayable low dynamic range images.

### 5. Mertens Exposure Fusion

Mertens Exposure Fusion is used as an alternative approach. Unlike Debevec and Robertson methods, Mertens Fusion does not require exposure time information. It directly combines useful regions from multiple exposure images into a displayable fused result.

### 6. CLAHE Contrast Enhancement

CLAHE is applied to the Mertens Fusion output as a post-processing step. The purpose of this step is to improve local contrast, especially in regions where details are less visible.

## Evaluation

The outputs are evaluated using both visual and quantitative comparisons. The following measurements are used:

- Mean brightness
- Contrast value based on grayscale standard deviation
- Histogram analysis

Accuracy is not used because this project is not a classification problem. PSNR and SSIM are also not used because there is no ground-truth HDR reference image for this experiment.

## Summary

This project shows that combining multiple exposure images can improve the visibility of details in high-contrast scenes compared to using a single exposure image. Among the tested methods, Mertens Fusion with CLAHE was used as the final enhanced output because it combines exposure information and improves local contrast.
